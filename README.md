# Homebrew Tap 

This is a custom Homebrew tap for my projects.

## Installation

Add the tap to your Homebrew installation:

```bash
brew tap felipe-veas/tap
```

## Formulae

### kpdbug

A powerful Kubernetes debugging tool.

Project URL: https://github.com/felipe-veas/k8s-pods-debug

```bash
brew install kpdbug
```

## Updating

To update formulae, run:

```bash
brew update
brew upgrade
```
